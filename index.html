Rocket Propulsion Applications

Rocket Propulsion Applications consists of uses and examples of different engines which are used at diiferent stages in a vehicle.

Booster

  • Provides initial propulsion thrust to launch the vehicle
  • Provides a large Thrust
  • It has a high thrust-weight ratio
  • It has a high specific impulse(Isp)
  • Examples
    • F-1
    • RD-170
    • SSME
    • RS-68

Sustainer(2nd stage)

  • Provides supplemental impulse for achieving orbit
  • Provides a high Thrust but lesser than the booster is needed
  • It has a higher area ratio than bosters
  • Examples
    • J-2
    • RD-120
    • LR91 series
    • NK-39

Upper Stage Engine(USE)

  • It is applies for final orbit insertion
  • It can be used even for modification of orbital parameters
  • It provides a low to medium thrust(10-300 kN)
  • The propellants whichare used are either hypergolic(spontaneously ignite when in contact with each other in fuels) or LO2/LH2
  • Examples
    • RL10 family
    • Agena
    • Vinci

Orbital Maneuvering Engine(OME)

  • It provides a moderate thrust(0.1-20 klbf) depending upon the vehicle mass and operational requirements
  • It has a long life and can handle multiple starts
  • It is mostly pressure fed hypergolic and it is radiatively cooled
  • It also provides a thrust on demand
  • Examples
    • AJ10-137 SPS
    • AJ10-190 OME
    • TRW VTE

RCS/ACS(Reaction Control System/Altitude Combustion Stand)

  • It provides thrust on demand and pulse mode
  • It is also throtteable(capable of having thrust varied)
  • It provides a low thrust(0.1-1000 lbf)
  • It is mostly pressure-fed monopropellant or a hypergolic bipropellant
  • Examples
    • Hughes HE-54
    • Bell Model 8414
    • Marquardt R-4D

Planetary Ascent/Descent

  • It provides low-medium thrust(0.6-10 klbf)
  • It is also throtteable(5%-10% power)
  • It provides a thrust on demand which has a high reliability with a fast response
  • Examples
    • Aerojet MR-80
    • TRW VTR-10 LMDE
    • Bell/Rkdn LMAE
